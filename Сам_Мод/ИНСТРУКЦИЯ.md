# Установка мода

1. Убедитесь, что игра **Pacific Drive** закрыта.
2. Скопируйте файл `ZZZZ_NOPRIDE_P.pak` из этой папки.
3. Откройте папку с игрой Steam и перейдите по пути:
   `[Ваша_Папка_Steam]\steamapps\common\Pacific Drive\PenDriverPro\Content\Paks\~mods\`
   *(Если папки `~mods` нет, просто создайте её).*
4. Вставьте файл `ZZZZ_NOPRIDE_P.pak` в папку `~mods`.
   *(Важно: если там уже есть старые версии мода вроде `NoPride_P.pak` или `zzz_NoPride_P.pak`, удалите их, чтобы избежать конфликтов).*
5. Запускайте игру и наслаждайтесь кастомными мемами в меню кастомизации универсала!
